# Travel Trucks

Travel Trucks is a camper rental catalog built with React, TypeScript, Redux Toolkit, and Vite. Users can browse campers, filter results, open a detailed camper page, save favorites, and send a booking request.

## Features

- Browse camper listings in a responsive catalog page
- Filter campers by location, form, engine, transmission, and equipment
- Save favorite campers with persistence in `localStorage`
- Open camper details with gallery, specifications, reviews, and booking form
- Load more campers from the catalog without leaving the page

## Installation

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

## Usage

- Open the home page at `http://localhost:5173`
- Use `View Now` to navigate to the catalog
- Apply filters and click `Search` to fetch matching campers
- Click the heart button to add or remove favorites
- Click `Show More` to open camper details in a new tab
- Submit the booking form with your name, email, and date

## Scripts

- `npm run dev` starts the Vite development server
- `npm run build` creates a production build
- `npm run lint` runs ESLint
- `npx prettier . --write` formats the project
